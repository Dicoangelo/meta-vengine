# Systematic Debug

Debug the issue using a methodical approach.

## Process
1. **Reproduce**: Understand exactly what's failing
2. **Isolate**: Find the minimal reproduction
3. **Trace**: Follow the data/control flow
4. **Hypothesize**: Form theories about root cause
5. **Test**: Verify each hypothesis
6. **Fix**: Implement the smallest correct fix
7. **Verify**: Confirm the fix works without side effects
